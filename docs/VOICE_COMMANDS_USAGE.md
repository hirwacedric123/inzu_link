## üéØ **How to Use**

### **Step-by-Step:**

1. **Activate Voice Commands**
   - Click the microphone button OR press 'V' key
   - Button will turn red and pulse when listening

2. **Speak Your Command**
   - Wait for "Listening..." message
   - Clearly say your command (e.g., "Go to home")
   - Speak naturally - no need to be overly precise

3. **Command Execution**
   - You'll see "Heard: [your command]" message
   - Command will execute automatically
   - Page will navigate to requested location

4. **Feedback**
   - Green message = Success
   - Red message = Error
   - Orange message = Processing

---

## üí° **Tips for Best Results**

1. **Speak Clearly** - Enunciate your words
2. **Quiet Environment** - Reduce background noise
3. **Wait for Confirmation** - Wait for "Listening..." before speaking
4. **Use Natural Language** - Say commands naturally
5. **Check Microphone** - Ensure microphone is working and not muted

## üé® **Visual Indicators**

### **Voice Button States:**

- **Green/Blue** - Ready to listen
- **Red (Pulsing)** - Currently listening

### **Feedback Messages:**

- **"Listening..."** - System is ready for your command

- **"Command not recognized"** - Try rephrasing or say "help"

## üîß **Troubleshooting**

### **"Microphone permission denied"**
- **Solution:** Go to browser settings and allow microphone access for this site
- **Chrome:** Settings ‚Üí Privacy ‚Üí Site Settings ‚Üí Microphone

### **"No speech detected"**
- **Solution:** 
  - Check microphone is connected
  - Speak louder or closer to microphone
  - Check microphone is not muted

### **"Voice recognition not supported"**
- **Solution:** Use Chrome or Edge browser for best support

### **Command not working**
- **Solution:**
  - Try rephrasing the command
  - Say "help" to see available commands
  - Check if you're using a supported command
